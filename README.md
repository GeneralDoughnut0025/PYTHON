SIMULATIONS
- HEAT TRANSFER SIMULATION ON A 1D OBJECT  :
This code simulates transient heat conduction in a 1D rod. Using a numerical finite-difference approach, it calculates how heat spreads from the hot ends into the cooler interior over time and visualizes the changing temperature profile as the system evolves. https://github.com/user-attachments/assets/6b435e86-a8d9-4a27-8ef0-fb7a4fed303d
- HEAT TRANSFER SIMULATION ON A 2D OBJECT :
This simulation models 2D heat transfer across a plate using numerical finite differences. It shows how heat spreads from the boundaries over time and how the temperature pattern evolves across the surface. https://github.com/user-attachments/assets/495ed1ef-1c95-4742-b6f2-d7eefcca2ae6
- DOUBLE PENDULUM SIMULATION : 
This simulation solves and animates the motion of a double pendulum using Lagrangian mechanics and numerical ODE integration. It captures the coupled, highly sensitive motion of the system over time. https://github.com/user-attachments/assets/be68598c-abf1-41fa-8da8-83543dca8355
- UNIVERSAL GAS CONSTAT SIMULATION :
This simulation visualizes ideal-gas behavior by tracking particles moving and colliding inside a closed container. It shows how random particle motion and wall collisions relate to basic gas-law behavior and collision-based pressure effects. https://github.com/user-attachments/assets/ce4fe8cb-b53f-4399-83f9-9745ba9ee03e
- D2Q9 LATTICE-BOLTZMAN MODEL :
This simulation uses the D2Q9 Lattice-Boltzmann Method to model 2D fluid flow around a circular obstacle. It visualizes how the flow field evolves over time, including wake formation and velocity patterns behind the cylinder. https://github.com/user-attachments/assets/dfc2abca-7d3f-4ef0-8673-781d74a846fb
- ENGINE CRANKSHAFT SIMULATION :
This simulation models single-cylinder crank-slider kinematics and animates piston motion over one engine cycle. It tracks piston position, velocity, acceleration and key geometric parameters like RPM, crank radius, and rod length can be varied in real time. https://github.com/user-attachments/assets/4bb84c18-bc6e-4298-a7e8-aebbcc13ffb7
